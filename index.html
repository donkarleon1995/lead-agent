🚀 3 уровня автоматизации агента
1
Ручной — браузер + Claude API
Ты листаешь ТенЧат, агент анализирует и пишет черновик
АКТИВЕН СЕЙЧАС Старт: 1-2 дня
▼
● Вы здесь — используй вкладку «Анализ поста»
Браузер (ТенЧат)
Claude API (этот интерфейс)
Google Sheets / Notion CRM
1
Открываешь ТенЧат в браузере
Ищешь по ключевым словам: «аутстафф java», «нужен аналитик», «ищем BA/SA», «нехватка ресурсов». Листаешь ~15-20 мин/день.
2
Копируешь пост в раздел «Анализ поста»
Вставляешь имя, должность, компанию и текст поста. Нажимаешь «Анализировать».
3
Claude скорит и пишет черновик DM
API вызов к claude-sonnet-4-6 → получаешь скор 0-100, разбивку по 5 критериям, список сигналов и готовый черновик.
POST /v1/messages { model: "claude-sonnet-4-6" } → { score: 88, tier: "hot", draft: "Мария, привет!..." }
4
Правишь черновик и копируешь в ТенЧат
Редактируешь при необходимости, копируешь → вставляешь в DM лиду в ТенЧат. Принятые лиды вручную заносишь в CRM.
✓ Плюсы
+Нет технической настройки — старт сегодня
+Полный контроль качества каждого лида
+Валидируешь критерии скоринга на реальных постах
+Стоимость Claude API ~$5-20/мес
✗ Минусы
–30-40 мин ручного труда в день
–Зависит от твоей дисциплины и расписания
–Не работает 24/7 без тебя
2
Полуавтомат — Browser Use + Python
Агент сам листает ТенЧат в твоей сессии, ты только проверяешь
СЛЕДУЮЩИЙ ШАГ Настройка: 1-2 недели
▼
Browser Use (Python)
Claude API
Telegram Bot
n8n / Make
Notion CRM
1
Установка Browser Use
Python-библиотека даёт LLM управление браузером через твои cookies — не нужен отдельный аккаунт ТенЧат.
pip install browser-use playwright playwright install chromium # Копируешь cookies из браузера → agent использует твою сессию
2
Агент обходит ТенЧат по расписанию
Cron каждые 2-4 часа: агент открывает ТенЧат, выполняет поиск по ключевым словам, листает результаты, извлекает посты.
task = "Открой tenchat.ru. Найди посты по запросам: 1) аутстафф java 2) нужен аналитик 3) ищем BA Извлеки: автор, должность, компания, текст. Верни JSON."
3
LLM-скоринг каждого поста через API
Каждый пост → Claude API с промптом скоринга. Посты score ≥ 60 идут в очередь BDM. Черновик DM генерируется автоматически.
4
Telegram-бот уведомляет BDM
Карточка лида: скор + сигналы + черновик. Три кнопки: ✅ принять / ✏️ изменить / ❌ пропустить. Принятые → Notion CRM.
✓ Плюсы
+80% рутины автоматизировано
+Мониторинг каждые 2-4 часа без тебя
+BDM только проверяет готовые лиды
✗ Минусы
–Нужны Python-навыки или разработчик
–Может сломаться при обновлении UI ТенЧат
–Нужен VPS для фонового запуска (~$10/мес)
3
Полный автомат — Claude Computer Use
Агент видит экран как человек — не зависит от структуры HTML
ПРОДВИНУТЫЙ Настройка: 2-4 недели
▼
Claude Computer Use API
Docker + VNC
Telegram Bot
VPS с GUI (~$20-40/мес)
1
Агент видит экран как скриншоты
Claude Computer Use делает скриншоты, анализирует визуально, кликает и скроллит как человек. Работает с любым UI — даже мобильным приложением в эмуляторе.
2
Полный цикл без парсинга DOM
Не нужно знать HTML-структуру ТенЧат. Агент читает текст со скриншота, понимает интерфейс и действует. Обновления UI не ломают агента.
instruction = "Открой ТенЧат. Найди посты про аутстафф java. Прочитай первые 10. Для score > 70 — открой профиль, скопируй данные, сообщи мне с черновиком DM."
3
Human-in-the-loop сохраняется
Агент никогда не отправляет сообщения самостоятельно — только готовит черновик и присылает в Telegram. BDM жмёт одну кнопку.
✓ Плюсы
+Не ломается при обновлении UI ТенЧат
+Работает с мобильным приложением
+Максимальная автономность 24/7
✗ Минусы
–Дороже: ~$0.5-1 за сессию по API
–Нужен VPS с графическим интерфейсом
–Сложнее настроить и отлаживать
